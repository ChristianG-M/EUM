Este componente se utiliza para recibir, validar y almacenar los billetes que ingresen. Se debe de considerar la corriente y potencia que consume pues hay picos de corriente que las fuentes usadas no puedan suministrar. Considerar la fuente de tension utilizada para este componente si existen problemas en el sistema.
Características:
	Supply Voltage * : 12V DC ±5% (Greater than 4.0A/50W) - 24V DC ±5% (Greater than 2.0A/50W)

	Current Consumption: 12V DC
	• Standby = 200mA
	• Operation = 2.3A
	• Peak = 3.0A

	Current Consumption: 24V DC
	• Standby = 120mA
	• Operation = 1.2A
	• Peak = 1.4A

	Maximum Current Limitation Mode 
	• Standby = 200mA
	• Operation = 1.4A
	• Peak = 2.0A

	Sleep Mode 
	• Mode A = 35μA (Typ)	
	• Mode B = 95μA (Typ)
