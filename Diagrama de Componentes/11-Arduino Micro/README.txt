Este componente se eligió cosiderando los puertos de entrada y salida que requerimos para todos los componentes, al menos debe de contar con:
-Un conector USB 2.0 / 3.0 para comunicación entre la tarjeta madre y el arduino
-Una entrada digital para sensor infrarrojo (sensor de presencia)
-Una entrada digital para un boton, este se utiliza para cancelar la operación en curso
-4 salidas digitales a relevador para ventiladores
-1 salida digital a relevador para magnetización de chapa
-3 salidas digitales a transistor para luces led
-Una entrada para interrupción, este se utiliza para detectar el estado de la chapa magnetica, si esta alimentada o no (en caso de utilizarse)*
-Una entrada para interrupción, este se utiliza para detectar la apertura o cierre de la puerta
-1 UART para comunicación MDB (billetero o monedero)
-1 SPI para comunicación CCTalk (Hopper's)

Total:
- 1 conector USB 2.0 / 3.0
- 10 pines digitales
- 2 pines de interrupción
- 1 UART
- 1 SPI
