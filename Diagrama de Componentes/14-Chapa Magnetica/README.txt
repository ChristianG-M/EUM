Este componente se encarga de mantener abierta o cerrada la puerta. Se recomienda que la chapa siempre se encuentre energizada (cerrada) con una bateria de respaldo dedicada solo para la chapa en caso de falta de enercía residencial. Dos pines son de alimentación y se interrumpe el positivo y otros dos son de continuidad, si la chapa esta energizada existe continuidad en dos pines, si no no existe la continuidad.
